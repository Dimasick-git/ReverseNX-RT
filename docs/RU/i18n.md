# Локализация ReverseNX-RT (Ryazhenka)

## Архитектура

```
Overlay/source/main.cpp
  └─ OverlayTest::initServices()
       ├─ tsl::hlp::doWithSmSession (resolve language):
       │    ├─ setInitialize()
       │    ├─ setGetSystemLanguage(&langCode)
       │    ├─ setMakeLanguage(langCode, &lang)
       │    └─ rnxs::loadLanguage("ru" | "en")
       └─ tsl::hlp::doWithSmSession (original SaltyNX init)
                │
                ▼
Overlay/source/rnxs_lang.cpp
  └─ loadFile("sdmc:/config/ReverseNX-RT/lang/<code>.json")
       ├─ cJSON_ParseWithLength()
       └─ для каждого ключа из kTable: *e.dst = item->valuestring
```

## Путь установки

```
/config/ReverseNX-RT/lang/en.json
/config/ReverseNX-RT/lang/ru.json
```

Релизный zip распаковывается в корень SD-карты и кладёт их туда автоматически.

## Маппинг языка системы

| HOS `SetLanguage` | 2-letter code | JSON-файл |
|-------------------|---------------|-----------|
| `SetLanguage_RU` | `ru` | `lang/ru.json` |
| любой другой | `en` | `lang/en.json` |

## Как добавить новый ключ

1. `Overlay/include/rnxs_lang.hpp` — `extern std::string NEW_KEY;`
2. `Overlay/source/rnxs_lang.cpp` — `std::string NEW_KEY = "default English";` + строка в `kTable`.
3. `lang/en.json`, `lang/ru.json` — добавить ключ с переводом.
4. `Overlay/source/main.cpp` — использовать как `rnxs::NEW_KEY` (если C++ string) или `rnxs::NEW_KEY.c_str()` (если C string).
